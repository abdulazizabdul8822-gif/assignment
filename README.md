1.ans:

aar:
old javaScript (ES5)
function scope - available throughout the function
can be redeclared with the same name

let:
modern JavaScript (ES6+)
block scope - only available inside if, for, while etc.
cannot be redeclared

const:
modern JavaScript (ES6+)
block scope - only available inside specific block
cannot be redeclared


2.

spread operator:
three dots (...)
expands/spreads array or object
used to copy or merge data


3. what is the difference between map(), filter(), and forEach()?
map():
transforms each element
returns new array
original array unchanged

filter():
selects elements based on condition
returns new array
only includes elements that pass the condition
forEach():

performs action on each element
returns nothing (undefined)
for execution/side effects only




4.
arrow function:

modern JavaScript (ES6+)
shorter and cleaner syntax
uses => symbol (looks like an arrow)


5.
template literals:

written with backticks (`)
insert variables using ${expression}
supports multi-line strings
